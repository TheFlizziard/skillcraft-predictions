About us and about the project
This project is an university project which constists of analyzing a given dataset with Python and answering a problematic.

As a team of two students, we worked on a dataset that contains datas about players statistics on the famous game Starcraft II, developped by Blizzard Entertainment. The dataset is made of 3395 observations and 20 variables, where LeagueIndex is the label and other variables are features.

The goal of this project is to try to predict the rank of any Starcraft II players based on their statistics on the game.
